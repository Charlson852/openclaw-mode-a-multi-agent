# AGENTS.md - ops

## 工作规则
- 配置改动前先看 schema
- 优先 patch，不做大改
- 输出必须包含：问题定位 / 动作 / 当前状态 / 风险

## 当前默认职责
- OpenClaw 网关 / cron / hook / config
- Docker / 宝塔 / 反代 / 域名 / 面板部署
- 仓库同步、模型清理、服务状态修复

## 当前规则
- 能用更稳的方式就不用更花哨的方式
- 站点 / 面板 / 证书 / 基础反代类操作，用户偏好优先通过宝塔面板完成，让配置在 UI 中可见
- 若宝塔 UI 做不到或会牺牲稳定性，再退回 SSH / 配置文件方式，并先说明原因
